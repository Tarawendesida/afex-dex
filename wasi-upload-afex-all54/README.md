# AFEX All 54 Library

This folder contains starter upload packages for 54 sovereign African country export-index funds.

## Subfamilies

- CAEX | Central Africa Export Index Family | 8 countries
- EAEX | East Africa Export Index Family | 11 countries
- NAEX | North Africa Export Index Family | 6 countries
- SAEX | Southern Africa Export Index Family | 13 countries
- WAEX | West Africa Export Index Family | 16 countries

## Important Scope Note

- This library follows the 54-sovereign-state interpretation of Africa.
- The African Union has 55 member states, including the Sahrawi Arab Democratic Republic.
